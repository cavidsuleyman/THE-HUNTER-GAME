## İSTİFADƏ QAYDASI


> **ADDIM-1:**   `git clone https://github.com/cavidsuleyman/MILLIONAIRE-GAME.git` <br/>
> **ADDİM-2:**  yaranan `MILLIONAIRE-GAME` qovluğuna daxil olun <br/>
> **ADDİM-3:**  içərisində olduğunuz qovluğu mətn redaktorunda açın <br/>
